# Free Fire Diamond Top Up

**Free Fire Diamond Top Up** একটি ওয়েবসাইট, যেখানে প্রতি সপ্তাহে ৫ জন র‍্যান্ডম বিজয়ীকে ফ্রি ডায়মন্ড দেওয়া হয়। ব্যবহারকারীরা তাদের নাম বা গেম আইডি দিয়ে অংশগ্রহণ করতে পারে।

## বৈশিষ্ট্য
- **নাম/গেম আইডি দিয়ে অংশগ্রহণ:** ইউজাররা নাম বা গেম আইডি দিয়ে প্রতি সপ্তাহে ফ্রি ডায়মন্ডের জন্য অংশগ্রহণ করতে পারেন।
- **র‍্যান্ডম বিজয়ী নির্বাচন:** প্রতি সপ্তাহে র‍্যান্ডমভাবে ৫ জন বিজয়ী নির্বাচন করা হয়।
- **সহজ ইন্টারফেস:** সিম্পল এবং ব্যবহারকারী-বান্ধব ডিজাইন।

## কীভাবে ব্যবহার করবেন
1. ওয়েবসাইটে গিয়ে তোমার নাম বা গেম আইডি ইনপুট করো।
2. "Enter" বাটনে ক্লিক করো।
3. "Pick Winner" বাটনে ক্লিক করলে, র‍্যান্ডমভাবে ৫ জন বিজয়ী নির্বাচন হবে এবং তাদের নাম দেখানো হবে।
